# retail-billing-system-python
A simple retail billing system using Python Tkinter to generate, save, and print bills for customers.
# 🧾 Retail Billing System

A GUI-based billing system built with Python.

## 🚀 Features
- Generate customer bill
- Calculate total with tax
- Save bill to file
- Search previous bills
- Print bill

## 🛠️ Technologies
- Python
- Tkinter
- File Handling

## ▶️ How to Run
python billing_system.py

## 📂 Folder
- bills/ (stores saved bills)
